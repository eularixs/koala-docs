Koala is a native macOS app for building, testing, and shipping HTTP APIs.
It's the Postman experience without Electron — keyboard-first, local-first,
and opinionated about the things that matter.

## Why a native client?

- **Fast.** Cold-start under 300 ms on M-series Macs.
- **Local.** Your data lives in `~/Library/Application Support/Koala`. Sync is opt-in.
- **Secure.** Secrets live in the macOS Keychain, not in a JSON file.
- **Calm.** No telemetry by default. No "AI assistant" popups.
