---

Koala speaks GraphQL natively. No "convert to GraphQL" mode.

## Connect

Point at any GraphQL endpoint:

```
https://api.example.com/graphql
```

Koala runs introspection in the background and populates the schema sidebar.

## Build queries

- **Autocomplete** as you type, with field descriptions inline.
- **Fragments** in a sidebar — drag to insert.
- **Variables** edited in a structured JSON pane.

## Subscriptions

Subscriptions just work. Koala opens a WebSocket and renders pushed
messages in the timeline view.
